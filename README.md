# garichi's init.el

## INSTALL
```sh
cd ~
git clone https://github.com/garicchi/.emacs.d.git
```

## PACKAGES
|name|description|
|:---:|:---:|
|package.el|melpa等のパッケージを使うやつ|
|auto-complete|コードの自動補完をしてくれるやつ|
|atom-dark-theme|atomっぽい配色テーマ|
|neotree|左側にディレクトリツリーを表示してくれる|
|flycheck|ソースコードのエラーをチェックしてくれる|
|web-mode|html内に埋め込まれたjsやcssをハイライトしてくれる|

## FOR MAC OS
システム設定から```キーボード```→```ショートカット```→```入ソース```→```前の入力ソースを選択```のチェックをはずす

## インデント
```sh
C-x space
# 矩形選択
C-t
# tab ot delete
```
